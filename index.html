The Problem

Circular Dependency

When resources are requested in a fixed order (Left → Right) by all threads, a Hold and Wait cycle forms. If P1-P5 all grab their left fork, the system enters an irreversible hang.

The Solution

Resource Hierarchy

By forcing the last philosopher to pick up the Right Fork first, we break the circular symmetry. This ensures that at least one resource is always available, preventing the system from freezing.
